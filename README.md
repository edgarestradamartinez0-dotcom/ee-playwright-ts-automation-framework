# EE Playwright QA Framework

A TypeScript + Playwright test automation framework covering UI and API testing.

## Stack
- Playwright
- TypeScript
- Allure Reporting
- GitHub Actions CI/CD

## Setup
\`\`\`bash
npm install
npx playwright install
\`\`\`

## Run Tests
\`\`\`bash
npx playwright test
\`\`\`
